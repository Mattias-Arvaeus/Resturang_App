# Resturang App
En "Enkel" app för Torento i Stenungsund. Ska göras med hjälp av följande verktyg;
* SASS
* Gulp / Grunt
* Bootstrap
* PhoneGap

Allt utom Phonegap kommer att intsalleras i [Node.js](https://nodejs.org/dist/v10.16.3/node-v10.16.3-x64.msi).

## Installera resurser
SASS
```bash
npm install -g sass
```
